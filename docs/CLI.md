## Quick Reference

```bash
# Run a workflow
sloth-runner run -f workflow.sloth

# List tasks
sloth-runner list -f workflow.sloth

# Stack management
sloth-runner stack list
sloth-runner stack run -f workflow.sloth --stack my-stack

# Agent management
sloth-runner agent start
sloth-runner agent status
```
