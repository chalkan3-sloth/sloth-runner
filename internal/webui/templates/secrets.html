Secrets Management

Encrypted secrets are managed via CLI only

Security Notice: For security reasons, secrets must be managed through the CLI using encrypted keys. The web UI only displays which secrets exist (names only), not their values.
CLI Commands
Add a secret:
sloth-runner secret add <stack-id> <secret-name> <secret-value>
List secrets for a stack:
sloth-runner secret list <stack-id>
Get a secret value:
sloth-runner secret get <stack-id> <secret-name>
Remove a secret:
sloth-runner secret remove <stack-id> <secret-name>
